# Football-analytics-Arsenal-Fc                               
 Objectif: Expliquer par la  donnée la chute d'Arsenal durant la dernière décénie
 Contenu : -Comprendre et appréhender la problématique posée                                          
-Collecte, nettoyage des données 
-Traitement et manipulations des données (Python/ SQL)
-Webscrapping/ Intégration des données et analyses
-Data Visualization/ Data storytelling
-Soutenance + création d'un rapport Word et d'un Dashboard avec l'outil Power BI du projet
 Résultat :  - Détection des tendances sur les performances 
-Identification des principaux facteurs de succès
-Influence du staff sur la capacité a maximiser les chances de victoires
-Recommandations financières / disciplinaire
